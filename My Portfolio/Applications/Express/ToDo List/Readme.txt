This application creates todo lists 
    - Uses bootstrap, node, express, ejs, MongodDB, and Mongoose

Features:
    - Create ToDo lists and add tasks
    - Delete ToDo lists
    - Delete tasks
    - Search by ToDo list title
    - View all created ToDo lists

Requirements:
    - This application uses mongoDB atlas: https://www.mongodb.com/atlas/database
    - userName: provided by mongoDB
    - password: set while creating mongoDB account
    - DB: the name of the database where the data will be stored
        - created automatically if selected name doesn't exist

Notes:
    - Posts are stored using mongoDB atlas and are persistant
